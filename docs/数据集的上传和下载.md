

从网盘授权直接下载到`/root/autodl-tmp`
然后移动到文件系统`/root/autodl-fs`

```
mv /root/autodl-tmp/* /root/autodl-fs/
```


在执行投影前的数据集结构确认
```sh
00004                          
    pylon_camera_node/                           # 存放 RGB 彩色图像 (.jpg)。
    pylon_camera_node_label_id/                  # 存放 2D 语义标签 (.png)。图像中每个像素的值对应一个类别 ID。
    pylon_camera_node_label_color/               # 存放 2D 标签的可视化图 (.png)。
    os1_cloud_node_kitti_bin/                    # 存放 原始激光雷达点云 (.bin)。二进制格式，包含 (x, y, z, intensity)。 
    os1_cloud_node_semantickitti_label_id/       # 存放 3D 点云语义标签 (.label)。
    transforms.yaml                # 定义传感器之间的 外参变换矩阵（通常是 LiDAR 到相机的变换），用于将点云投影到图像上。
    poses.txt                      # 机器人的 轨迹/位姿数据。用于将每一帧局部点云拼接到全局地图中。
    calib.txt                      # 相机 内参（焦距、光心等）和标定信息
    camera_info.txt                # 相机 内参（焦距、光心等）和标定信息

compress_rellis.sh            # 个人用于压缩数据集的脚本，实际项目使用不到
Rellis_3D_cam_intrinsic
Rellis_3D_cam2lidar_20210224
说明.txt  
```
